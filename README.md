# Web-Dev-Project

DataBridge is a simple web application built with Angular and Django. The frontend provides a user-friendly interface, while the backend handles data processing and communication with the database.

The application allows users to create, view, edit, and delete data through a REST API. Angular is used for dynamic page updates, and Django manages the server logic and data storage.

This project demonstrates the basics of full-stack development and interaction between frontend and backend systems.
